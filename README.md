## рҹҡҖ Getting Started

### Prerequisites

Make sure you have these installed:

```bash
node --version   # Must be v18.0.0 or higher
npm --version    # Must be v8.0.0 or higher
git --version    # Any recent version
```

### Installation

**Step 1 вҖ” Clone the repository**
```bash
git clone https://github.com/yourusername/swiftcv.git
cd swiftcv
```

**Step 2 вҖ” Install dependencies**
```bash
npm install
```

**Step 3 вҖ” Set up environment variables**
```bash
cp .env.example .env.local
```
Then fill in your API keys (see [Environment Variables](#-environment-variables) below).

**Step 4 вҖ” Set up Supabase database**

Run these SQL commands in your Supabase SQL editor:

```sql
-- Profiles table
CREATE TABLE profiles (
  id UUID REFERENCES auth.users(id) PRIMARY KEY,
  full_name TEXT,
  email TEXT,
  target_role TEXT,
  college TEXT,
  grad_year TEXT,
  ai_rewrites_used INTEGER DEFAULT 0,
  ai_rewrites_date DATE,
  created_at TIMESTAMPTZ DEFAULT now()
);
ALTER TABLE profiles ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Users own profile" ON profiles FOR ALL USING (auth.uid() = id);

-- Resumes table
CREATE TABLE resumes (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id) ON DELETE CASCADE,
  title TEXT DEFAULT 'My Resume',
  data JSONB NOT NULL DEFAULT '{}',
  ats_score INTEGER DEFAULT 0,
  share_token TEXT UNIQUE DEFAULT gen_random_uuid()::text,
  is_public BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);
ALTER TABLE resumes ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Users own resumes" ON resumes FOR ALL USING (auth.uid() = user_id);

-- Resume views tracking
CREATE TABLE resume_views (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  resume_id UUID REFERENCES resumes(id) ON DELETE CASCADE,
  viewed_at TIMESTAMPTZ DEFAULT now(),
  ip_hash TEXT
);
```

**Step 5 вҖ” Run the development server**
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. рҹҺү

## рҹҡў Deployment

### Deploy to Vercel (Recommended)

**Step 1** вҖ” Push your code to GitHub:
```bash
git add .
git commit -m "feat: SwiftCV ready for deployment"
git push origin main
```

**Step 2** вҖ” Go to [vercel.com](https://vercel.com) вҶ’ New Project вҶ’ Import from GitHub

**Step 3** вҖ” Add environment variables in Vercel dashboard (same as `.env.local`)

**Step 4** вҖ” Update Supabase Auth settings:
- Go to Supabase вҶ’ Authentication вҶ’ URL Configuration
- Add Site URL: `https://your-domain.vercel.app`
- Add Redirect URL: `https://your-domain.vercel.app/auth/callback`

**Step 5** вҖ” Deploy! рҹҡҖ

```bash
# Or deploy directly via CLI
npm install -g vercel
vercel --prod
```
